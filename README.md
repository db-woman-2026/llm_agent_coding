# LLM Agent Coding

LLM coding agent의 특징과 현명한 사용 방법을 다루기 위한 강의 준비 저장소입니다.

현재는 Codex를 실제 프로젝트에 사용하면서 남긴 요청 사례를 모아, 강의 주제와 실습 흐름을 설계하기 위한 원자료를 정리하고 있습니다.

## 문서

- [사용자 요청 모음](docs/asks.md)

## 목적

- 실제 요청 패턴을 기반으로 LLM coding agent 활용 방식을 분석합니다.
- 좋은 요청과 애매한 요청의 차이를 강의 자료로 정리합니다.
- 프로젝트 분석, 코드 수정, 검증, git/GitHub 작업, 장기 작업 관리 사례를 강의 흐름으로 재구성합니다.
