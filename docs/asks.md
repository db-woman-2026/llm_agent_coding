~~~text
```text
TODO
password
error
```

컨셉:

```text
내 컴퓨터 안의 문서를 대상으로 직접 검색 도구를 만드는 느낌

예시:

```text
90일 이상 수정되지 않은 .zip 파일 목록

```

컨셉:

```text
실제 삭제는 하지 않고 후보만 보여주면 안전하면서도 실용적인 자동화가 됨
```

---

```
~~~
